This is a privacy policy statement for app Dream Catcher.

<b>Explaining permissions:</b>

android.permission.VIBRATE maxSdkVersion=18
This permission is needed for devices with Android version up to 4.3, so vibration can be controlled when  notificaion is triggered.
Permission isn't requested for devices beyond 4.3.

android.permission.ACCESS_NETWORK_STATE
For billing and basic usage analytics.

android.permission.INTERNET
For in app purchases and analytics reporting.

android.permission.RECEIVE_BOOT_COMPLETED
So notifications can get set up when device starts.

android.permission.USE_FINGERPRINT [23+]
For fingerprint authentication.

com.android.vending.BILLING
For in app purchases.


<b>What we know about you</b>

App sends basic analytics, so we can identify basic user behaviour.
We don't collect or store any data that can identify any user.
